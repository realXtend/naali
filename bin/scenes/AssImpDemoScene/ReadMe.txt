This scene demonstrates the OpenAssetImporter. Collada or other Assimp suported files
can be used instead of Ogre .mesh files. Just put the collada file as a mesh ref to EC_Mesh component.

TentDome.dae and house2.dae are from google warehouse: http://sketchup.google.com/3dwarehouse/
More information about the jeep1.ms3d is found from jeep1.readme.txt.
